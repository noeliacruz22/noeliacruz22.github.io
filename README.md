## ⚡ Funcionalidades

- Navegación lateral fija con detección de sección activa
- Efecto typewriter en el Hero — roles rotativos
- Animaciones de entrada al hacer scroll (`IntersectionObserver`)
- Hover interactivo en pills, cards y chips de stack
- Formulario de contacto con envío real a Gmail
- Diseño 100% responsive
- Línea de tiempo de formación académica
- Grid de publicaciones estilo Instagram
- Cards de proyectos con overlay al hover
